# Competitive Analysis Framework

> How to analyze competing games and identify market opportunities.

---

## Analysis Template

### Game Analysis
```markdown
## [Game Name]

### Basic Info
- **Genre:** [Genre]
- **Platform:** [Web/Mobile/Steam]
- **URL:** [Link]
- **Release date:** [Date]
- **Price:** [Free/Paid/PWYW]
- **Revenue:** [If known]

### Core Loop
- **Action:** [What player does]
- **Feedback:** [What happens]
- **Reward:** [What player gets]
- **Motivation:** [Why player continues]

### Scoring (out of 5)
- Art direction: [1-5]
- Core loop: [1-5]
- Juice/polish: [1-5]
- Progression: [1-5]
- Retention: [1-5]
- Monetization: [1-5]

### What Works
- [Strength 1]
- [Strength 2]
- [Strength 3]

### What Doesn't
- [Weakness 1]
- [Weakness 2]
- [Weakness 3]

### Opportunities
- [Gap 1]
- [Gap 2]
- [Gap 3]
```

### For Each Competitor
- [ ] Played for 30+ minutes
- [ ] Identified core loop
- [ ] Scored across 6 categories
- [ ] Documented strengths
- [ ] Documented weaknesses
- [ ] Identified gaps

### For Each Genre
- [ ] Researched top 5 games
- [ ] Analyzed market size
- [ ] Identified trends
- [ ] Found gaps
- [ ] Assessed competition level

## Sources

### Data Sources
- Steam: Store trends, reviews, player counts
- itch.io: Trending games, sales data
- Poki: Featured games, play counts
- CrazyGames: Popular games, categories
- Google Trends: Search interest over time
- Social Media: What's being shared

### Research Methods
- Play top 10 games in genre
- Read player reviews (positive and negative)
- Watch gameplay videos
- Monitor social media discussions
- Track download/play counts
- Analyze monetization models

## Competitive Advantage Framework

### Differentiation Axes
1. **Visual style** — Unique art direction
2. **Core loop** — Different mechanics
3. **Platform** — Browser vs mobile vs desktop
4. **Monetization** — Free vs paid vs PWYW
5. **Audience** — Casual vs core vs niche
6. **Scope** — Small vs medium vs large

### Positioning Map
```
                    Casual
                      |
                      |
        [Our Game]    |    [Competitor A]
                      |
                      |
   Simple ────────────┼──────────── Complex
                      |
                      |
        [Competitor B]|    [Competitor C]
                      |
                      |
                    Core
```

### Questions to Answer
1. Who is our target player?
2. What need do we serve?
3. How are we different?
4. Why would players choose us?
5. What's our unfair advantage?
