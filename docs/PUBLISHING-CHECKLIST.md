# Publishing Checklist — Browser Games

> Step-by-step guide for publishing AI Game Studio games.
> To be updated with research findings.

---

## Platform Requirements

### CrazyGames
- [ ] Create developer account
- [ ] Read SDK documentation
- [ ] Integrate HTML5 SDK
- [ ] Add rewarded ads
- [ ] Add midgame ads
- [ ] Test ad placement
- [ ] Submit for review
- [ ] Wait for approval (1-3 days)

**Requirements:**
- HTML5, responsive
- No external requests
- Minimum 5 minutes of gameplay
- No adult content

### Poki
- [ ] Contact Poki team
- [ ] Submit game for review
- [ ] Implement Poki SDK (if required)
- [ ] Optimize for engagement metrics
- [ ] Launch

**Requirements:**
- High engagement metrics
- Polished gameplay
- Mobile-friendly

### Itch.io
- [ ] Create developer account
- [ ] Set pricing (PWYW)
- [ ] Prepare supporter extras
- [ ] Write game description
- [ ] Create screenshots/trailer
- [ ] Publish

**Requirements:**
- None (any game allowed)
- PWYW pricing recommended

### Newgrounds
- [ ] Create developer account
- [ ] Read API documentation
- [ ] Integrate Newgrounds API
- [ ] Submit for review
- [ ] Wait for approval

**Requirements:**
- Flash or HTML5
- No adult content

## Pre-Launch Checklist

### Game Quality
- [ ] Core loop is clear and satisfying
- [ ] No critical bugs
- [ ] Performance is acceptable (60fps on mid-range)
- [ ] Mobile-friendly (responsive)
- [ ] Sound works
- [ ] Save/load works (if applicable)

### Documentation
- [ ] Game description written
- [ ] Screenshots taken (5-10)
- [ ] Trailer recorded (60-90 seconds)
- [ ] Press kit created
- [ ] Controls documented

### Technical
- [ ] Build optimized (minified, compressed)
- [ ] No console errors
- [ ] No external dependencies (or bundled)
- [ ] Analytics integrated (if planned)
- [ ] Ads integrated (if planned)

### Marketing
- [ ] Landing page live
- [ ] Social media accounts set up
- [ ] Devlog started
- [ ] Email list ready
- [ ] Press contacts identified

---

## Launch Checklist

### Day Before
- [ ] Final build tested
- [ ] All platform submissions ready
- [ ] Social media posts scheduled
- [ ] Email blast prepared
- [ ] Launch day schedule planned

### Launch Day
- [ ] Publish on all platforms
- [ ] Post launch announcement
- [ ] Send email blast
- [ ] Monitor for issues
- [ ] Respond to feedback
- [ ] Celebrate!

### Day After
- [ ] Check metrics
- [ ] Respond to reviews
- [ ] Fix critical bugs
- [ ] Thank supporters
- [ ] Plan first update

## Post-Launch Checklist

### Week 1
- [ ] Monitor player metrics
- [ ] Respond to all feedback
- [ ] Fix critical bugs
- [ ] Post updates on social media
- [ ] Start planning next game

### Week 2-4
- [ ] Release first update
- [ ] Share patch notes
- [ ] Continue social media posting
- [ ] Expand to new platforms (if successful)
- [ ] Build community

### Month 2+
- [ ] Add new features
- [ ] Optimize based on data
- [ ] Start next game

## Common Issues

### Build Fails
- Check console for errors
- Test in multiple browsers
- Verify all assets are bundled

### Ads Not Showing
- Check SDK integration
- Verify ad unit IDs
- Test in development mode

### Performance Issues
- Reduce particle count
- Optimize textures
- Use LOD for 3D models

### No Players
- Improve marketing
- Get featured on platforms
- Share on social media
- Ask friends to play
