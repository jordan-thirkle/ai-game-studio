## Concept Details

### Game 2: 3D Survivor-like
**Inspiration:** Vampire Survivors, Brotato, 20 Minutes Till Dawn
**Our twist:** 3D top-down, floating islands, skyfarer theme
**Core loop:** Move → kill → level up → new ability → more killing
**Monetization:** Rewarded ads (revive), cosmetic skins
**Why browser-first:** Most survivor-likes are Steam ports. Browser-native 3D version doesn't exist.

### Game 3: Idle/Incremental
**Inspiration:** Cookie Clicker, Melvor Idle, Adventure Capitalist
**Our twist:** Skyfarer theme, automate island exploration
**Core loop:** Build → automate → prestige → rebuild
**Monetization:** Premium unlock, cosmetic themes
**Why browser:** Idle games are browser royalty. Millions play on Poki/CrazyGames.

### Game 4: 3D Puzzle/Exploration
**Inspiration:** Blue Prince, The Witness, Outer Wilds (lite)
**Our twist:** Mystery puzzles in floating sky ruins
**Core loop:** Explore → find clue → solve puzzle → unlock area
**Monetization:** Premium levels, hint system
**Why browser:** 3D puzzle games are rare in browser. Blue Prince validated demand.

### Game 5: Roguelite Deckbuilder
**Inspiration:** Balatro, Slay the Spire, Monster Train
**Our twist:** 3D cards, skyfarer theme, elemental combos
**Core loop:** Fight → choose reward → build deck → harder fight
**Monetization:** Expansion packs, cosmetic cards
**Why Steam:** Deckbuilders monetize best on Steam ($10-20 price point).
