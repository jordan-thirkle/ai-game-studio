# Contributing

AI Game Studio is open source. We believe in building in public.

## How to Contribute

### Fork a Game
1. Fork the game repo (e.g., whisperwood-v2)
2. Make improvements
3. Open a PR with before/after screenshots
4. Describe what you changed and why

### Improve the Pipeline
1. Fork this repo
2. Update the game-director pipeline or scoring rubric
3. Open a PR with evidence of improvement

### Add a Game
1. Build a game using the 7-phase pipeline
2. Add it to `src/data/games.ts`
3. Add screenshots to `public/images/`
4. Open a PR

## Code of Conduct

- Be honest in scoring — inflated scores help no one
- Document what you learn — the journey is the content
- Share what works — the flywheel spins faster with more contributors
